LD Grace Time Feature to Odoo Hrms

This Module provides grace time configuration
and functionality to your Odoo ERP.

Usage of this module


1. Install this module.

2. Go into Attendances module > configuration
and fill "Grace Time" value.

3. Now system will compute the status of employee if he is running after grace time, System will mark him as half-day.
